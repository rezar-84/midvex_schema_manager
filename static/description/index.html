Midvex Schema Manager

JSON-LD Structured Data for Odoo 19 Websites

Overview

Midvex Schema Manager is a general-purpose Odoo 19 module for managing JSON-LD structured data across one or multiple Odoo websites. It complements Odoo's native SEO tools (meta title, description, keywords) by injecting <script type="application/ld+json"> blocks into every page head automatically.

Key Features

  • Global Organization and WebSite schema per website
  • Page-specific and custom-URL schema records
  • Language-specific schema records (separate record per language)
  • 12 built-in system templates: Organization, WebSite, Product, FAQPage, BreadcrumbList, Article, BlogPosting, Service, ContactPage, AboutPage, CollectionPage, ItemList
  • Custom template support — create, duplicate, import, export, deactivate
  • Manual JSON override (admin only) — validated on save, never concatenated
  • Auto-fill from page SEO metadata (meta title, description, canonical URL)
  • Internal validation: @context, @type, required fields, URL formats, language codes, Product / FAQPage / BreadcrumbList rules
  • Duplicate schema detection with warnings
  • QWeb injection into website.layout head
  • Schema preview endpoint at /schema-preview/<id> for admins
  • Links to Google Rich Results Test and Schema Markup Validator

Schema Types

Organization WebSite Product FAQPage BreadcrumbList Article BlogPosting Service ContactPage AboutPage CollectionPage ItemList

Where to Find It

After installation, navigate to Website → Configuration → Structured Data.

Security

  • Schema Manager: full create / edit / delete access (inherits Website Publisher)
  • Schema User: read-only access (inherits Website Designer)
  • All JSON output uses json.dumps() — never string concatenation
  • Manual JSON is parsed and validated before saving
  • Inactive schemas and wrong-language schemas are never rendered

Compatibility

  • Odoo 19 Enterprise (primary target)
  • Odoo 19 Community (compatible)
